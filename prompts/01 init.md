# Senv

这是一个提供私密配置、环境变量加密存储和解密的程序。

# 功能

1. 提供多个功能：环境变量、配置文件管理
2. 数据保存在指定路径（类似于配置项目），并加密

- 每个路径下的文件有单独的密码，由用户指定
- 用户在数据密码后，将密码加密后存储起来作为每次使用时的解密密钥

3. 程序提供项目初始化功能，指定路径、密码，默认路径 ~/.config/senv/data
4. 环境变量管理：

- 环境变量分 group，哪些 group 生效由配置文件决定
- 提供 get/set/list 功能
- 可以在 shell rc 文件快速解密、导入环境变量

5. 配置文件管理功能：

- 支持创建文件，可指定恢复后的文件保存路径
- 提供 editor 功能：解密后用 vim 编辑；保存后自动加密并放到对应位置
- 支持导出，指定配置名，导出到配置的路径，也可以重新指定
- 支持 list
